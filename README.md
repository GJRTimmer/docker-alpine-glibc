# Installation

<details>
<summary>Install from DockerHub</summary>
<p>

Download:
```bash
docker pull datacore/alpine-glibc:latest
```

Build:
```bash
docker build -t datacore/alpine-glibc https://github.com/GJRTimmer/docker-alpine-glibc
```
</p>
</details>
